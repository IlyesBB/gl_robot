Déplacement vers une balise (cible)¶

class gl_lib.sim.robot.strategy.deplacement.balise.TournerVersBalise(**kwargs)[source]¶

Stratégie qui permet de tourner le robot robot vers une balise devant lui, si elle est visible Pour cela, on utilise les images capturée pour la caméra du robot

get_angle()[source]¶

Retourne 1 si la balise est à droite, -1 si elle est à gauche

Si l’angle formé par le vecteur formé par le milieu bas de l’écran et le centre de la balise avec la verticale est supérieur à 5 degrés, la balis est considérée devant le robot on retourne alors 0

Si aucune balise trouvée retourne None :return: int
